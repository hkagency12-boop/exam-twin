# Exam Twin

## Beyond Marks: Measuring Understanding, Not Just Scores

### Problem
Traditional examinations focus mainly on marks and fail to measure a student's actual understanding and confidence level.

### Solution
Exam Twin is an AI-inspired learning analysis system that evaluates both performance and confidence to generate a Learning Twin Report.

### Features
- Score Analysis
- Confidence Analysis
- Knowledge Index
- Learning Twin Report
- Personalized Recommendations

### Target Users
- Students
- Teachers
- Schools

### Theme
Examinations

### Future Scope
- AI-powered analysis
- Topic-wise weakness detection
- Learning path recommendations
- School dashboard integration
